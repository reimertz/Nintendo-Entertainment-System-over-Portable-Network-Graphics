Why?

This is a tribute to the old times, when iframes was the new big thing and JavaScript wasn't widely used.

I wanted to investigate how hard it would be to implement a <noscript>-compatible nes emulator that could be played together with other people in realtime. A stretch goal was to make it work in browsers from the late 90's.

Ironically, everything is written in JavaScript on the server using Node.js. :D

The design of this page is a tribute to first page ever created, check it out here.
